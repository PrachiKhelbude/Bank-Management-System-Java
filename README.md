# Bank Management System (Java)

A Java Swing + MySQL based ATM / Bank Management System project.

## Features
- Login using Card Number & PIN
- Secure authentication (JDBC)
- Signup and Transactions
- Java Swing GUI

## Technologies
- Java
- MySQL
- JDBC
- Swing

## How to Run
1. Clone the repository
2. Configure MySQL database
3. Run Login.java
